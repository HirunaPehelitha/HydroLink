# Hydro Link — Cinematic Hydration Experience

![Hydro Link Logo](assets/Asset%201%20(1).png)

A high-fidelity, industrial-minimalist landing page for **Hydro Link**, the next generation of smart hydration. This project features a cinematic design, smooth scroll-driven animations, a real-time HUD (Heads-Up Display) for specifications, and a unique collaboration with the world of **AVATAR**.

## 🚀 Live Demo
Experience the site here: **[https://hirunapehelitha.github.io/HydroLink/](https://hirunapehelitha.github.io/HydroLink/)**

## ✨ Key Features
- **Cinematic UI**: Premium dark-mode aesthetic with vibrant lime accents.
- **Interactive HUD**: 360° specification display with floating tech compartments.
- **Scroll-Driven Animation**: Realistic bottle reveal as you explore the site.
- **Avatar Collaboration**: Immersive Pandora-inspired full-width visual experience.
- **Atmospheric Audio**: Featuring the *Interstellar* theme by Hans Zimmer for a truly cinematic journey.
- **AR Ready**: Integrated QR code for instant AR model viewing.
- **Responsive Design**: Optimized for everything from ultra-wide monitors to mobile devices.

## 🛠️ Technology Stack
- **HTML5** (Semantic structure)
- **Vanilla CSS3** (Custom animations, glassmorphism, flex/grid)
- **JavaScript** (Scroll logic, music toggles, animation triggers)

## 🎨 Design System
- **Colors**: Matte Black, Lime Green (`#ADFF00`), Pure White.
- **Typography**: Montserrat (Modern, Bold, High-readability).
